# Premium UI Development Workflow

This workflow ensures the consistent generation of high-quality, memorable frontend interfaces, integrating the `frontend-design` skill in Antigravity.

When triggered to create or redesign an interface, strictly follow these steps:

1. **Analyze Requirements and Intent**
   - Understand the user's specific goals, target audience, and any technical constraints.
   - Ask clarifying questions if the UI's purpose or core functionality is unclear.

2. **Load the Frontend Design Skill**
   - Read the `.agents/skills/frontend-design/SKILL.md` document to contextualize the advanced aesthetic guidelines.

3. **Propose the Design Direction**
   - **Do NOT write any code yet.**
   - Formulate 1 to 2 distinct visual concepts based on the guidelines (e.g., a "Neon Maximalist" option vs. an "Editorial Minimalist" option).
   - Pitch the typography pairings, color palette, and micro-animation strategy.
   - Wait for the user's explicit approval or directional feedback.

4. **Blueprint and Build Core Structure**
   - Once approved, begin implementation by creating the structural assets:
     - `index.html` (Semantic structure).
     - `index.css` or framework styling (CSS variables, reset, font imports).
   - Implement the foundational background textures, base typography, and layout grid.

5. **Component Integration and Refinement**
   - Build out individual sections/components according to the bold design vision.
   - Break standard grid mentalities. Add depth (shadows, overlaps, asymmetric spaces).
   - Avoid generic placeholders where possible. Generate mock images or use descriptive SVG shapes.

6. **Polish and Animate**
   - Apply staggered entrance animations on load.
   - Add hover micro-interactions, custom cursors, or smooth transitions.
   - Double-check for non-generic aesthetic rules (No Inter/Arial, no flat uninspired backgrounds).

7. **Verification & Walkthrough**
   - Start any dev servers or compile routines if required.
   - Invite the user to preview the resulting UI and provide a walkthrough of the specific design choices made.
